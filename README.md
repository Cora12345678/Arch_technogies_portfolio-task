# Areeba Sajjad Portfolio Website

This project is a modern personal portfolio website designed to showcase the professional profile, skills, experience, and projects of Areeba Sajjad. The website provides an interactive and visually appealing interface to highlight web development, digital marketing, and technical expertise.

## Features

- Responsive and modern portfolio design
- Hero section with introduction and typewriter animation
- About section with statistics and professional summary
- Education timeline
- Work experience timeline with multiple roles
- Skills and technologies with progress bars
- Featured projects with links to GitHub and live demos
- Articles and insights related to SEO, WordPress, and technology
- Contact form for direct communication
- Social media links
- AI chatbot assistant for quick portfolio information
- Interactive UI elements and animations

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Three.js (for background animation)
- Google Fonts
- Responsive design techniques
